# LinuxCheatsheet
Commands I frequently use but forget and have to google


##### Push ssh-key to server
```
ssh-copy-id -i ~/.ssh/mykey user@host
```
##### After reinstallation of ssh on server
```
ssh-keygen -R [server ip]
ssh-keygen -R [server dns]
```
